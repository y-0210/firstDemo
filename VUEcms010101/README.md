# 一个基于vue-cli 使用axios作为http库的VUE移动端Store程序
最近在做一个vue的项目 做个demo
- 下载之后先安装依赖包
- 终端下运行 npm install
- 安装完成后运行 npm run serve 即可

## 主要使用的库
- webpack
- vue-router
- axios
- Vuex
- Vant移动端组件库
- Vue preview plugin
 
##功能
- 新闻资讯
- 图片分享
- 商品购买
- 购物车功能
